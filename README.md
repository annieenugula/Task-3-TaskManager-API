# Task Manager API (Django REST Framework)

A simple Task Manager API built using Django and Django REST Framework.  
This project allows users to create, view, update, and delete tasks through RESTful APIs.

---

## 🚀 Features

- Create new tasks
- View all tasks
- Retrieve single task details
- Update existing tasks
- Delete tasks
- REST API built with Django REST Framework

---

## 🛠️ Tech Stack

- Python
- Django
- Django REST Framework
- SQLite (default database)
