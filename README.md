# Task Manager API (Django REST Framework)

A simple Task Manager API built using Django and Django REST Framework.

---

## 🚀 Features
- Create tasks
- View tasks
- Update tasks
- Delete tasks

---

## 🛠 Tech Stack
- Python
- Django
- Django REST Framework
- SQLite

---

## ⚙️ Setup Instructions

### 1. Clone repo
git clone https://github.com/annieenugula/Task-3-TaskManager-API.git

### 2. Install dependencies
pip install django djangorestframework

### 3. Run migrations
python manage.py migrate

### 4. Run server
python manage.py runserver
